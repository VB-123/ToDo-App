# The API Docs
Here are the routes defined for the API.
## The base url 
Displays "Hello World!"
## To get all todos
**GET** : https://baseurl/todos
## To add todos
**POST** : https://baseurl/add-todo
## To edit a todo
**PUT** : https://baseurl/edit-todo/:id
## To delete a todo
**DELETE** : https://baseurl/delete-todo/:id
